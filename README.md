# matlabDrawLib
一些matlab绘图文件工具
### drawOnePeakCrosstalk
适用于绘制单峰、并求串扰率的图片
### drawTwoPeaksSBR
适用于绘制双峰、并求SBR的图片

### parseXML

适用于解析.xml文件数据，返回一个struct

### drawSpectrum

适用于用一个.xml文件绘制一条光谱

（程序中需要的x，y如何从parseXML.m返回的struct中提取可能需要更改……）

### drawMultiSpectrum

适用于用多个.xml文件数据绘制多条光谱

（程序中需要的x，y如何从parseXML.m返回的struct中提取可能需要更改……）
